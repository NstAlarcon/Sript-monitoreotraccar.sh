# Sript-monitoreotraccar.sh
Este script proporciona un sistema de monitoreo avanzado para Traccar, capaz de detectar fallos, reiniciar automáticamente el servicio, enviar alertas por WhatsApp, bloquear dispositivos GPS flooders y hasta reiniciar todo el servidor si es necesario.  Ha sido diseñado para entornos de producción donde la estabilidad es crítica
